I TAG HTML

STRUTTURA A panino

Sebbene nel corso degli anni siano state rilasciate varie versioni, le basi HTML rimangono le stesse. La struttura di un documento HTML è stata confrontata con quella di un sandwich. Poiché un sandwich ha due fette di pane, il documento HTML ha tag HTML di apertura e chiusura. Questi tag, come il pane in un panino, circondano tutto il resto
La struttura del documento a panino si costruisce con i tag html

ESEMPIO STRUTTURA A sandwich Subito dopo il tag HTML di apertura, troverai l'intestazione del documento, che è identificata dai tag head di apertura e chiusura. L'intestazione di un file HTML contiene tutti gli elementi non visivi che contribuiscono a far funzionare la pagina.

BODY TAG

ESEMPIO The Tag Il tag del corpo segue il tag della testa. Tutti gli elementi visivo-strutturali sono contenuti all'interno del body tag. Intestazioni, paragrafi, elenchi, citazioni, immagini e collegamenti sono solo alcuni degli elementi che possono essere contenuti all'interno del tag body.
